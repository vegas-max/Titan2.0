# OmniArb Full Matrix - Token Bridge Routes (Encoder/Decoder Build Sheet)

## Matrix Format
Each line represents a potential arbitrage route with the following fields:
- chain_origin: Source blockchain ID
- chain_dest: Destination blockchain ID  
- native_token: Token symbol being arbitraged
- dex_origin: DEX on source chain
- dex_dest: DEX on destination chain
- bridge_protocol: Cross-chain bridge to use
- liquidity_score: Estimated liquidity (0-100)
- fee_tier: Bridge/swap fee percentage
