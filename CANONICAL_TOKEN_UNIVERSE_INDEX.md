## Token Validation Criteria

### ERC-20 Compliance Checks

All tokens in the index are validated for:

1. **Contract Existence** - Verified on blockchain explorer
2. **ERC-20 Interface** - Implements required functions:
   - `totalSupply()`
   - `balanceOf(address)`
   - `transfer(address, uint256)`
   - `transferFrom(address, address, uint256)`
   - `approve(address, uint256)`
   - `allowance(address, address)`
3. **Decimals** - Proper decimals() function implementation
4. **Symbol** - Valid symbol() function
5. **Name** - Valid name() function
6. **Checksum Address** - All addresses are EIP-55 checksummed

### Validation Script

```python
from web3 import Web3

def validate_erc20_token(w3: Web3, token_address: str) -> bool:
    """
    Validate ERC-20 compliance for a token
    """
    # Check if address is checksummed
    if not Web3.is_checksum_address(token_address):
        return False
    
    # Minimal ERC-20 ABI
    erc20_abi = [
        {
            "constant": True,
            "inputs": [],
            "name": "decimals",
            "outputs": [{"name": "", "type": "uint8"}],
            "type": "function"
        },
        {
            "constant": True,
            "inputs": [],
            "name": "symbol",
            "outputs": [{"name": "", "type": "string"}],
            "type": "function"
        },
        {
            "constant": True,
            "inputs": [],
            "name": "name",
            "outputs": [{"name": "", "type": "string"}],
            "type": "function"
        }
    ]
    
    try:
        contract = w3.eth.contract(
            address=Web3.to_checksum_address(token_address),
            abi=erc20_abi
        )
        
        # Try to call standard functions
        decimals = contract.functions.decimals().call()
        symbol = contract.functions.symbol().call()
        name = contract.functions.name().call()
        
        # Basic validation
        if decimals is None or symbol is None or name is None:
            return False
        
        if decimals < 0 or decimals > 18:
            return False
        
        return True
        
    except Exception as e:
        print(f"Validation failed for {token_address}: {e}")
        return False
```

## Token Addition Process

### Steps to Add New Token

1. **Verify Contract**
   - Check on blockchain explorer (Polygonscan, Etherscan, etc.)
   - Confirm ERC-20 compliance
   - Verify liquidity on DEXes

2. **Validate Address**
   ```python
   from web3 import Web3
   checksum_address = Web3.to_checksum_address(token_address)
   ```

3. **Add to Registry**
   - Update this document
   - Add to config.json
   - Update token discovery scripts

4. **Test Integration**
   - Run validation script
   - Test with DexPricer
   - Verify quantum optimizer compatibility
